This file indicates the current version of each driver, and the release history for the Pump Driver Update. This update is compatible with Enabler v2.5x and higher.

Command Line Options:

  • /S - Selects a completely silent installation (must be the first command-line option).
  • /INSTANCE:NamedInstance - Set this option if the database is a named instance.
  • /USER:UserName - Pass the user name to connect to the database if required.
  • /PASSWORD:Password - Pass the password to connect to the database if required.
  • /? - Display Command Line Options.

Note:

  • This update does not always include the latest versions, due to beta testing or experimental versions. If you have received a driver from us to fix a current or recent problem, please check the DLL version before installing this update.
  • We advise any sites using these protocols to check Enabler Card Firmware, you may NEED to upgrade to the firmware in this Update.
    • Safe (CNG dispensers)
    • Sanki 2005 (SPD, MPD and Mobile dispensers)
    • Tatsuno SS-LAN
    • Tokheim UDC (with WWC computer)
    • Wayne DART
    • KwangShin
    • Censtar / Eaglestar
  • See articles about how to check firmware versions: Enabler PCI and Enabler Express
  • See articles about firmware updates on Enabler PCI and Enabler Express
  • Price multiplier functionality requires Enabler v3.40.26 or later.
  • Tank Gauge Drivers (Fafnir and Veeder Root) require Enabler v3.51 and later.
  • Tag Controller Drivers (MPI Technology) require Enabler v3.80.3 and later.
  • Orpak over TCP\IP requires Enabler v4.3.5 and later.
